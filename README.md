# 🛒 Retail Sales Forecasting & Inventory Optimization

An end-to-end **Data Science & Supply Chain Analytics project** that forecasts retail demand and optimizes inventory using ML, time-series modeling, and an interactive dashboard.

---

## 🎯 Problem

Retail businesses face:
- Overstocking (high cost)
- Stockouts (lost revenue)
- Uncertain demand

---

## 💡 Solution

This system:
- 📈 Forecasts future demand (ML + Prophet)
- 📦 Calculates safety stock & reorder points
- 🚨 Generates inventory alerts
- 📊 Visualizes insights via Streamlit dashboard

---

## 🧠 Key Features

- Demand Forecasting (Random Forest + Prophet)
- Inventory Optimization (Safety Stock, Reorder Point)
- Risk Classification & Reorder Recommendation
- Anomaly Detection & Revenue Estimation
- Interactive Dashboard

---

## ⚙️ Tech Stack

Python • Pandas • Scikit-learn • Prophet • Matplotlib • Streamlit
